快速入门#

创建web项目,导入servlet坐标

<dependency>
    <groupId>javax.servlet</groupId>
    <artifactId>javax.servlet-api</artifactId>
    <version>4.0.1</version>
    <scope>provided</scope>
</dependency>

<!-- Tomcat 10 使用这个-->
<dependency>
    <groupId>jakarta.servlet</groupId>
    <artifactId>jakarta.servlet-api</artifactId>
    <version>5.0.0</version>
    <scope>provided</scope>
</dependency>

简单使用例

package com.eoezora.controller;

import jakarta.servlet.ServletException;
import jakarta.servlet.annotation.WebServlet;
import jakarta.servlet.http.HttpServlet;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;

import java.io.IOException;

@WebServlet("/test") // 访问路径是 /test
public class TestController extends HttpServlet {
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        System.out.println("get");
    }

    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        System.out.println("post");
    }
}

urlPattern配置#

Servlet想要被访问,就必须设置其访问路径 (urlPattern)

一个Servlet可以配置多个urlPattern, 例如 @WebServlet(urlPattern = {"/test1", “/test2”, “/test3”})

urlPattern配置规则

优先级: 精确路径匹配 > 目录路径匹配 > 扩展名匹配 > / /*

精确匹配#

@WebServlet("/user/info") //访问路径为 -> http://localhost:8080/webapp/user/info

目录匹配#

以下面例子来说,只要在/user目录下的请求都能访问该Servlet

@WebServlet("/user/*") //访问路径为 -> http://localhost:8080/webapp/user/aaa | http://localhost:8080/webapp/user/bbb

拓展名匹配#

访问该拓展名的文件资源

@WebServlet("*.do") //访问路径为 -> http://localhost:8080/webapp/aaa.do

任意匹配#

不管访问地址是啥，都能访问这个Servlet

@WebServlet("/")
@WebServlet("/*") 
//访问路径为 -> http://localhost:8080/webapp/abc 可以用任意路径访问这个Servlet
//当我们在项目中设置了 "/" 和 "/*" 后会覆盖掉tomcat默认的DefaultServlet.当其他的urlPattern都匹配不上时会访问这个Servlet

xml配置Servlet#

需要在web.xml中配置, 例子如下, Servlet类与注解方式一样，只是不需要写注解@WebServlet

<servlet>
    <!-- servlet名字 -->
    <servlet-name>test</servlet-name>
    <!-- servlet类的路径 -->
    <servlet-class>com.eoezora.controller.TestController</servlet-class>
</servlet>

<servlet-mapping>
    <!-- servlet名字 -->
    <servlet-name>test</servlet-name>
    <!-- servlet访问路径 -->
    <url-pattern>/test</url-pattern>
</servlet-mapping>
