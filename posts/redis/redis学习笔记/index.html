数据类型#

Redis存储的时key-value结构的数据, 其中key是字符串类型, value有常用的5种数据类型

数据类型
string字符串普通字符串
hash哈希适合存储对象
list列表按照插入顺序排序, 可以重复
set集合无序集合, 没有重复元素
sorted set有序集合有序集合, 没有重复元素

常用命令#

字符串(string)操作#

SET [key] [value] # 设置指定key的值
GET [key] # 获取指定key的值
SETEX [key] [seconds] [value] #  设置指定key的值, 并设置过期时间 seconds 秒
SETNX [key] [value] # 只有key不存在时设置key的值

哈希(hash)操作#

HSET [key] [field] [value] # 设置哈希表 key 中的 field 的值为value
HGET [key] [field] # 获取哈希表中指定字段[field]的值
HDEL [key] [field] # 删除哈希表中的指定字段
HKEYS [key] # 获取哈希表所有字段
HVALS [key] # 获取哈希表中所有值
HGETALL [key] # 获取哈希表中所有字段跟值

列表(list)操作#

LPUSH [key] [value...] # 插入一个或多个值到列表头部
LRANGE [key] [startIndex] [stopIndex] # 获取指定范围内的元素
RPOP [key] # 移除并获取列表最后一个元素
LLEN [key] # 获取列表长度
BRPOP [key1] [key2] [TIMEOUT] # 移除并获取列表最后一个元素, 如果列表没有元素, 则阻塞列表直到等待超时, 或发现可弹出元素为止

集合(set)操作#

SADD [key] [member...] # 向集合添加一个或多个成员
SMEMBERS [key] # 返回集合中的所有成员
SCARD [key] # 获取集合成员数
SINTER [key...]# 返回给定所有集合交集
SUNION [key...] # 返回给定所有集合并集
SDIFF [key...]. # 返回给定所有集合差集
SREM [key] [member1] [member2] # 移除集合中的一个或多个成员

有序集合(sorted set)操作#

ZADD [key] [score1] [member1] [score2] [member2]... #向有序集合添加/更新一个或多个成员及分数
ZRANGE [key] [startIndex] [stopIndex] [withsScores] # 通过索引区间返回有序集合中指定区间内的成员
ZINCRBY [key] increment [menber] # 有序集合中对指定成员的分数加上增量 increment
ZREM [key] [member...] # 移除有序集合中的一个或多个成员

通用命令#

KEYS [正则表达式] # 查找符合给定表达式的key
EXISTS [key] # 检查给定key是否存在
TYPE [key] # 返回给定key所存储值的类型
TTL [key] # 返回给定key的剩余生存时间, 单位: 秒
DEL [key] # 当给定key 存在时删除key

在Java 中使用Redis#

SpringDataRedis#

maven坐标

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-redis</artifactId>
</dependency>

RedisTemplate#

SpringDataRedis中提供了一个高度封装的类: RedisTemplate, 针对jedis客户端中大量api进行归类封装, 将同一类型操作封装成operation接口, 具体如下

  • ValueOperations: 简单的K-V 操作
  • SetOperations: set类型数据操作
  • ZSetOperations: zset类型数据操作
  • HashOperations: 针对map类型的数据操作
  • ListOperations: 针对list类型的数据操作
    @Autowired
    private RedisTemplate redisTemplate;

	//通过redisTemplate.opsForXxxx()的方式获取操作对象
    void testRedis() {
        //获取ValueOperations并存值
        redisTemplate.opsForValue().set("test-123", "测试数据");
        //取值
        Object o = redisTemplate.opsForValue().get("test-123");
    }
